# ergonomic_keyboard_endgame
Making my own split keyboard, from scratch
## Goal:
- I want an ergonomic keyboards with:
    - Split Design
    - Tenkeyless (80%) Layout
    - Backlight
    - Raised
- Nice to have:
    - Fully wireless
    - Charging through cable
- Enjoy the learning
